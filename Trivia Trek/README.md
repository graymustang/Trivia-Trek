# Trivia Trek

This is a browser based trivia game that gets questions from the Open Trivia Database (https://opentdb.com/).
The player can choose a difficulty and a category, and answer multiple choice questions before the timer end. 
At the end, there is a screen showing the summary of the game.

## Tech Stack
- HTML
- CSS
- Basic JavaScript, no frameworks
- Open Trivia DB public api

## How to run
Download repository, then open "index.html" in your browser.

## Features
- Select difficulty (easy, medium, hard)
- Select a category
- Automatic question fetching from OpenTDB
- Shuffle the answers
- 15 second timer per question
- Tracking score
- Summary screen with rank
- Play again button to restart game
